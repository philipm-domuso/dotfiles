dotfiles
========

Config files and system bootstrap.

## How to bootstrap it

### On a *nix computer

```
git clone https://github.com/philipmat/dotfiles .dotfiles
cd .dotfiles/
source install.sh
```


### On a Windows computer

Using PowerShell:

```
git clone https://github.com/philipmat/dotfiles .dotfiles
cd .dotfiles/
./install.ps1
```
